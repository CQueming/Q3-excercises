HTML

HyperText Markup Language

Standard markup language to be used for web pages. Consists of series of elements and tags that are used to define the structure and content of a webpage.Htmml files are seen by web browsers to display the file to other users.

CSS

Cascading Style Sheets

CSS, or Cascading Style Sheets, is a language used to define the presentation and layout of a webpage written in HTML. It allows developers to separate the presentation of a webpage from its structure, making it easier to maintain and update. CSS can be used to specify the layout, color, font, and other design elements of a webpage.

Bootstrap

Bootstrap Version 4

Bootstrap is a popular open-source front-end development framework for creating responsive and mobile-first websites. It includes a collection of CSS and JavaScript components such as forms, buttons, and navigation, which can be easily integrated into a webpage. Bootstrap is widely used by developers because it makes it easier to create consistent and responsive designs across different devices and screen sizes.
